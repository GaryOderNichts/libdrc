New modules¶

  • Added support for Wii mode with one new application, aplmain_wiiactive and one new service, sc_wii.
  • Added support for Background mode with one new application, aplmain_background and one new service, sc_cmd_background.
  • Added a driver for another LCD panel from Panasonic: lcd_mc838985.
  • Added a driver for another I2C controller: i2c_samsung. Because the previous I2C driver was Renesas specific, it has been made generic and the Renesas driver was moved to the new i2c_renesas.
